# Barathraj A | Personal Portfolio

A modern, responsive, and visually appealing personal portfolio website designed to showcase my journey as an aspiring Data Scientist and Full Stack Developer.

## 🚀 Features

- **Responsive Design:** Fully styled to adapt to any screen size, providing a seamless experience on desktop, tablet, and mobile, utilizing Bootstrap 5.
- **Interactive UI:** Features smooth scrolling navigation, hover effects, and modern CSS styling (`bg-blobs`, glassmorphism nav).
- **Dedicated Sections:**
  - **Home:** A welcoming hero section introducing me.
  - **About:** A brief background on who I am.
  - **Skills:** A grid showcasing my technical expertise in Programming (Python, JS), Data Science (ML, Data Analysis), Web Dev, and Tools.
  - **Projects:** Highlights of my featured work including "Smart Helmet," a "Hackathon Solution," and "Data Analysis Suite."
  - **Contact:** A functional contact form and links to my professional profiles.

## 🛠️ Technologies Used

- **HTML5:** Semantic and accessible structure.
- **CSS3:** Custom stylesheet (`style.css`) combined with **Bootstrap 5** for layout and utility classes.
- **JavaScript:** For interactive elements and smooth behaviors (`script.js`).
- **FontAwesome:** For intuitive icons.
- **Google Fonts:** Utilizing 'Inter' and 'Outfit' typography.

## 💻 Getting Started

Running this project locally is incredibly simple. You do not need any build steps or local servers.

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone https://github.com/itsmebarath/your-repo-name.git
   ```
2. **Navigate to the directory**:
   ```bash
   cd "barath portfolio"
   ```
3. **Open `index.html`**:
   Simply double-click the `index.html` file to open it in your preferred web browser.
